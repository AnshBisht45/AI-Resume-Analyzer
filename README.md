🧠 AI Resume Analyzer :
An AI-powered full-stack web application that analyzes resumes, extracts skills, and calculates ATS (Applicant Tracking System) score using FastAPI and React.

🚀 Live Demo :
🌐 Frontend: (https://ai-resume-analyzer-9dzzc5cr0-ansh-bisht-s-projects.vercel.app)
⚙️ Backend: (https://ai-resume-analyzer-cdsk.onrender.com)

📌 Features :
📄 Upload Resume (PDF)
🧠 Extract text from resumes automatically
🔍 Skill detection using keyword matching
📊 ATS Score calculation
🎨 Clean and responsive UI
⚡ Real-time analysis results

🛠 Tech Stack :
Frontend:
React.js
HTML, CSS, JavaScript

Backend:
FastAPI
Python

Libraries:
pdfplumber

Deployment:
Vercel (Frontend)
Render (Backend)

⚙️ How It Works :
User uploads a resume (PDF)
Backend extracts text using pdfplumber
Skills are matched from predefined skill list
ATS score is calculated based on skill matches
Results are sent back to frontend and displayed
